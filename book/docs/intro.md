# ٹیوٹوریل کا تعارف

آئیے **Docusaurus کو 5 منٹ سے بھی کم وقت میں** دریافت کرتے ہیں۔

## آغاز کرنا

**ایک نئی سائٹ بنا کر** شروع کریں۔

یا **فوری طور پر Docusaurus کو آزمائیں** **[docusaurus.new](https://docusaurus.new)** کے ساتھ۔

### آپ کو کیا ضرورت ہوگی

- [Node.js](https://nodejs.org/en/download/) ورژن 20.0 یا اس سے اوپر:
  - Node.js انسٹال کرتے وقت، یہ تجویز کیا جاتا ہے کہ آپ تمام چیک باکسز کو نشان زد کریں جو انحصار سے متعلق ہیں۔

## ایک نئی سائٹ بنائیں

**کلاسک ٹیمپلیٹ** کا استعمال کرتے ہوئے ایک نئی Docusaurus سائٹ بنائیں۔

کمانڈ چلانے کے بعد کلاسک ٹیمپلیٹ خود بخود آپ کے پروجیکٹ میں شامل ہو جائے گا:

```bash
npm init docusaurus@latest my-website classic
```

آپ اس کمانڈ کو Command Prompt، Powershell، Terminal، یا اپنے کوڈ ایڈیٹر کے کسی بھی دوسرے مربوط ٹرمینل میں ٹائپ کر سکتے ہیں۔

کمانڈ Docusaurus کو چلانے کے لیے درکار تمام ضروری انحصار بھی انسٹال کرتی ہے۔

## اپنی سائٹ شروع کریں

ڈیولپمنٹ سرور چلائیں:

```bash
cd my-website
npm run start
```

`cd` کمانڈ اس ڈائریکٹری کو تبدیل کرتی ہے جس کے ساتھ آپ کام کر رہے ہیں۔ اپنی نئی بنائی گئی Docusaurus سائٹ کے ساتھ کام کرنے کے لیے، آپ کو ٹرمینل کو وہاں نیویگیٹ کرنے کی ضرورت ہوگی۔

`npm run start` کمانڈ آپ کی ویب سائٹ کو مقامی طور پر بناتی ہے اور اسے ایک ڈیولپمنٹ سرور کے ذریعے پیش کرتی ہے، جو آپ کو http://localhost:3000/ پر دیکھنے کے لیے تیار ہے۔

`docs/intro.md` (یہ صفحہ) کھولیں اور کچھ لائنیں ترمیم کریں: سائٹ **خود بخود ری لوڈ ہوتی ہے** اور آپ کی تبدیلیاں دکھاتی ہے۔
